# 🌿 Plant Disease Detector

An ML-powered application that detects plant diseases from images using a trained deep learning model.

## 🚀 Features
- Detects multiple plant diseases from leaf images
- Trained deep learning model (.h5)
- Real-time image capture and prediction
- Web-based interface using Flask

## 🛠️ Tech Stack
- Python
- TensorFlow / Keras
- Flask
- OpenCV
- NumPy

## 📁 Project Structure
- `app.py` — Main Flask application
- `plant_detector.py` — Disease detection logic
- `plant_disease_model.h5` — Trained ML model
- `classes.npy` — Disease class labels

## 🔧 How to Run
```bash
pip install -r requirements.txt
python app.py
```
